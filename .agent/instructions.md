# Project Instructions: Videoplayer

This project is a web-based video player application.

## Development Workflow
1. **Component Creation**: All new UI components must be added to a `src/components` directory.
2. **Styling**: Use Vanilla CSS for all components. Keep `index.css` for global variables and layout.
3. **Testing**: Run `npm test` after adding new features (to be configured).

## Agent Guidelines
- Always prioritize semantic HTML and accessibility.
- Follow the design patterns established in existing files.
- Consult `.agent/rules` for specific coding standards.
